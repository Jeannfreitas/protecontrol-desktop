# 💻 ProteControl Desktop

Aplicação desktop do sistema **ProteControl**, desenvolvida para facilitar o controle de trabalhos protéticos diretamente no computador, sem depender de navegador.

Essa versão foi criada com foco em praticidade no uso diário dentro de laboratório, oferecendo uma interface simples, rápida e offline.

---

## 🚀 Sobre a aplicação

O **ProteControl Desktop** é uma versão desktop do sistema, permitindo ao usuário gerenciar pedidos, dentistas e informações importantes de forma local.

---

## ⚙️ Funcionalidades

- 📋 Visualização e gerenciamento de pedidos  
- 🦷 Controle de dentistas  
- 🛠 Organização de tipos de trabalho  
- 💰 Acompanhamento de valores e pagamentos  
- 📝 Registro de observações dos trabalhos  
- ⚡ Interface rápida e leve  
- 💻 Execução como aplicativo desktop (sem navegador)  

---

## 🧠 Como funciona

O sistema é empacotado com Electron, que transforma a aplicação web em um programa desktop.

Isso permite:

- Rodar sem navegador  
- Instalar no computador  
- Utilizar como software comum
